Reading


Bootstrap is one of the most popular front-end web development frameworks.Bootstrap offers many of its advantages such as being responsive, cross-browser compatible, easy to use, and free.

Bootstrap was originally developed by Twitter.

Bootstrap includes multiple classes that can be used to style virtually any element on a page such as text, forms or images, a 12 column grid system, and various reusable components such as buttons, modals, navigation, and more.

There are multiple versions of Bootstrap available. As a general rule, you want to use the latest stable version (currently version 4).

Bootstrap is a library, and as such you need to add it to your site before you can use it. There are two ways: Download Bootstrap from the official website OR include links from a CDN (Content Delivery Networks). For a CDN, you will need several files including CSS and JavaScript files

The Bootstrap website contains a starter temple which includes the needed links. You can find it here

The Reboot CSS rules allow for adding classes and styling various components on the page. It is easy to add them by adding a class attribute to the element. For example, the Bootstrap colors classes can be found here

The Bootstrap grid system consists of 12 columns. Several grid classes can be used, along with breakpoints, to create layouts for different device sizes. The structure of the Bootstrap grid consists of rows and columns. You can find the Bootstrap grid rules and options here
